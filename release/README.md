#Ionide-VSCode: F#

It's part of [Ionide](http://ionide.io) plugin suite.
F# IDE-like possibilities in Atom editor and Visual Studio Code

## Features

- Better syntax highlighting
- Auto completions
- Error highlighting and error list
- Tooltips
- Go to Declaration
- Show symbols in file
- Highlighting usages
